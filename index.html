Hexo使用

发表于 2019-08-24 更新于 2019-09-02 分类于 hexo

安装

  1. 下载并安装git
    git

  2. 下载并安装node.js
    Node.js
    在命令行下执行命令 node -v
    输出如下信息：
    v10.16.3

  3. 安装Hexo
    执行命令 npm install -g hexo-cli
    执行命令 hexo -v来确认安装成功
    安装发布工具 npm install hexo-deployer-git --save

使用

  1. 创建blog

后面的blog为名字，可以任意。本地会创建一个文件夹

  1. 生成文件，并启动服务

其中hexo generate可以简写成hexo g, hexo server 可以简写成hexo s

  1. 在浏览器查看已经创建的服务器
    打开地址 http://localhost:4000

发布到Github

  1. 创建一个github账号
    github

  2. 新建一个repo
    名字必须为 xxx.github.io, 其中xxx为github账号名称

  3. 修改blog目录下的站点配置文件_config.yml的配置项deploy

注意冒号后面有一个空格

  1. 发布到服务器上

其中hexo deploy可以简写成hexo d

  1. 打开浏览器查看服务器
    打开地址 http://xxx.github.io

写新文章

  1. 新增加一个文章

这样就在blog/source/_posts新增加一个文件hexo使用.md的文件

  1. 进入目录修改上述文件

    不要删除前面自动生成的东西

  2. 重新生成并发布到服务器上

  3. 确认新文档发布成功
